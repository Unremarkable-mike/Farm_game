# 🌾 Simple 2D Farm Game Prototype

A small pixel art farming game prototype built with the Godot Engine. This project serves as a basic proof of concept for farming mechanics and provides a foundation for expanding into a complete farming game.

## Features

- 🌱 Basic farming gameplay
- 🎨 Pixel art graphics
- 🕹️ Player movement
- 🚜 Simple farm environment
- ⚙️ Built with Godot Engine

## Getting Started

### Requirements

- Godot Engine (version used for development)

### Running the Project

1. Clone this repository:

```bash
git clone https://github.com/yourusername/farm-game-prototype.git
```

2. Open the project in Godot.

3. Press **F5** or click **Run Project**.
